Mēs esam ļoti ieinteresēti, lai uzzinātu, kādi dati par atklātajiem pārtikas faktiem tiek izmantoti. Tas nav obligāts, bet mēs to ļoti augstu novērtētu, ja jūs pastāstītu mums par savu atkārtotu izmantošanu, lai mēs varētu tos dalīties ar sabiedrības Open Food Facts kopienu.

Open Food Facts datu eksporti

Datu bāzu izgāztuves un eksports tiek ģenerēti katru nakti.

Informācija par dažādiem MongoDB datu un CSV eksporta laukiem ir pieejama vietnē https://world.openfoodfacts.org/data/data-fields.txt.

MongoDB izgāstuve

Visu produktu dati ir pieejami MongoDB datu bāzē.

Saite: https://world.openfoodfacts.org/data/openfoodfacts-mongodbdump.tar.gz

sha256sum: https://world.openfoodfacts.org/data/sha256sum

md5sum: https://world.openfoodfacts.org/data/md5sum

CSV datu eksports

Visu produktu vai dažu produktu datus var lejupielādēt CSV formātā (lasāmi ar OpenOffice, Excel un daudzām citām programmām), izmantojot izvērstās meklēšanas formu .

Saite: https://world.openfoodfacts.org/data/en.openfoodfacts.org.products.csv

Faila kodēšana ir Unicode UTF-8. Laukumā atdalošais simbols ir <tab> (tabulācija).

RDF datu eksports

Datubāze ir pieejama arī RDF formātā. paziņojums franču valodā .

Saite: https://world.openfoodfacts.org/data/en.openfoodfacts.org.products.rdf

Eksperimentālā JSON API

JSON API ir pieejams arī produkta datu nolasīšanai. Šis API jo īpaši tiek izmantots iPhone un Android lietotnē Open Food Facts.

Šis API ir eksperimentāls un var mainīties. Lūdzu, pastāstiet mums, ja to izmantojat, lai mēs varētu jums pastāstīt par iespējamām izmaiņām.

Produkta datu nolasīšanas URL: https://world.openfoodfacts.org/api/v0/product/ [barcode] .json

Piemērs: https://world.openfoodfacts.org/api/v0/product/737628064502.json

Android un iPhone mobilā lietotne

Mobilo lietotņu Open Food Facts kods ir pieejams github . Lietotne ļauj lietotājiem skenēt produktu svītrkodu, apskatīt informāciju par produktu un uzņemt un nosūtīt attēlus un datus par trūkstošiem produktiem.

Programma ir izstrādāta HTML un Javascript, pateicoties Phonegap (Cordova), kas ļauj izmantot to pašu kodu visām mobilajām platformām.

Datu, API un eksporta adresātu saraksts

Jūs varat arī pievienoties adresātu sarakstam, lai apspriestu API un eksportu, pastāstītu mums par savu atkārtotu izmantošanu un saņemtu paziņojumu par API uzlabojumiem vai izmaiņām. Nosūtīt tukšu e-pastu uz api-subscribe@openfoodfacts.org abonēt.
