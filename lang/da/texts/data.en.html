Open Food Facts-dataekport

Databaseudtræk og -eksport genereres hver nat.

Oplysninger om de forskellige felter til MongoDB-udtræk og CSV-eksport er tilgængelig via https://world.openfoodfacts.org/data/data-fields.txt

MongoDB-udtræk

Data til alle produkter er tilgængelig i et MongoDB-databaseudtræk.

Link: https://world.openfoodfacts.org/data/openfoodfacts-mongodbdump.tar.gz

link: https://world.openfoodfacts.org/data/sha256sum

md5sum: https://world.openfoodfacts.org/data/md5sum

CSV-dataeksport

Data til alle produkter, eller til visse af produkterne, kan downloades i CSV-format (læsbare med OpenOffice, Excel og megen anden software) via avanceret søgeformular.

Link : https://world.openfoodfacts.org/data/en.openfoodfacts.org.products.csv

Filkodningen er Unicode UTF-8. Skilletegnet mellem felter er <tab>(tabulering).

Eksperimentel JSON API

En JSON API er også tilgængelig til læsning af produktdataene. Denne API benyttes især i mobil-app'en Open Food Facts-app til iPhone og Android.

Denne API er eksperimentel og kan ændre sig. Oplys os venligst om du bruger det, således at vi kan informere dig om evt. ændringer.

URL til at læse et givent produkts data: https://world.openfoodfacts.org/api/v0/product/[barcode].json

Eksempel: https://world.openfoodfacts.org/api/v0/product/737628064502.json

Android og iPhone mobil-app

Koden til mobil-app'en Open Food Facts-app er tilgængelig fra github. App'en giver brugerne mulighed for at skanne et produkts stregkode for at se produktinformationen samt for at tage og sende fotos og data for manglende produkter.

App'en er udviklet i HTML og Javascript takket være Phonegap (Cordova), der gør det muligt at benytte den samme kode til alle mobilplatforme.

Mailliste for data, API og eksport

Du kan også tilmelde dig mailinglisten for at drøfte API og eksport, fortælle os om dine genanvendelser og blive underrettet, når der foretages forbedringer eller ændringer til API'en. Send en tom e-mail til api-subscribe@openfoodfacts.org for at tilmelde dig.
